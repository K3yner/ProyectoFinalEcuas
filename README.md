# ProyectoFinalEcuas

En este proyecto se pretende utilizar la inteligencia artificial (chatGPT, copilot, deepseek y otras) para desarrollar programas que aproximen soluciones de ecuaciones diferenciales con métodos numéricos.

## Métodos escogidos
- Método de Rugen-Kutta de orden 4
- Método de Euler

## Tipos de ecuaciones a resolver

- ED de primer orden
    - Caída con fricción 
- ED de segundo orden 
    - Circuito RLC
- Sistema 2x2 Lineal
    - Resortes acoplados
